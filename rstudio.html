Global Vehicle Production and Propulsion Modeling

Modeling production volumes and propulsion trends using RStudio

This project applies RStudio for data cleaning, visualization, and trend modeling of global vehicle production by region and propulsion type (ICE, BEV, Fuel Cell, PHEV, FHEV, MHEV, REEV) to analyze output and create visual forecasts that track shifts between gas, hybrid, and electric propulsion production.

R will be used to forecast production changes over time and evaluate powertrain adoption rates. Future versions will include forecasting models to predict long-term propulsion trends.
